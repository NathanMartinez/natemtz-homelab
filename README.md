# Nathan Martinez Home Lab

Personal infrastructure and home lab repository for building a portfolio focused on systems administration, Linux, Docker, networking, reverse proxy, monitoring, and self-hosted services.

## Goals

- Build a clean, reproducible home lab
- Practice infrastructure as code principles
- Document services and architecture clearly
- Publish a professional portfolio site
- Support learning for DevOps, Sysadmin, and SRE-style roles

## Current Scope

- Portfolio placeholder site
- Cloudflare domain and Pages integration
- Planned service-by-service rollout

## Planned Services

- Reverse proxy
- DNS filtering / recursive DNS
- Monitoring and dashboards
- Media and utility services
- Documentation and diagrams

## Notes

This repository is being built incrementally. Public-facing services will be exposed selectively and only after they are documented, secured, and tested.
